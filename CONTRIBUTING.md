# How To Contribute

## Installation

* `git clone https://github.com/domoticalabs/ember-mqttjs.git`
* `cd ember-mqttjs`
* `npm install`

## Linting

* `npm run lint:hbs`
* `npm run lint:js`
* `npm run lint:js -- --fix`
